# Fireside Hacks

Invite all Web3 apps to burn UBI tokens.

## Install

```shell
npm i
```

## Test

```shell
npx hardhat test
```

## Roadmap

- Create UBI liquidity pools on every networks
- Deploy a UBI Burner contract to each network
- Invite all Web3 apps to integrate with UBI
